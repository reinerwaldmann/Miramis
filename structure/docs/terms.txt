===Термины и определения===
входной отчёт inreport - файл отчёта, получаемый со стенда могут именоваться результатами
выходной отчёт outreport - файл или html форма группового отчёта
протокол protocol - внутреннее представление протокола
результат result - внутреннее представление результата

===Описание базы данных===
Имя БД - MiramisDB
ПОльзователь - user
пароль - 123

Таблицы
-Таблица протоколов-
protocols
Поля:
    ID
    ProductName
    TestName
    pickle - поля для pickle-форм классов
    xml - поле для xml - формы представления
    text - поля для текстового представления


-Таблица результов-
results
Поля:
    ID
    ProductName
    TestName - имя программы испытаний
    Operator - имя оператора
    Date - дата испытаний
    SerialNumber - серийный номер
    BatchNumber - номер партии
    TypeOfTest - тип испытаний (предъявительские и т. п.)
    pickle - поля для pickle-форм классов
    xml - поле для xml - формы представления
    text - поля для текстового представления

-Таблица созданных отчётных форм-
outreports
Поля:
    ID
    ProductName
    TestName
    pickle - поля для pickle-форм классов
    xml - поле для xml - формы представления
    text - поля для текстового представления
